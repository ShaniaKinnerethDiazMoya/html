Tipos de Textos

En HTML, puedes utilizar varias etiquetas para dar formato a tus textos. Algunas de las más comunes son:

1. Texto normal

Este es un texto normal sin ningún formato especial.

2. Texto enfatizado

El texto enfatizado se puede lograr utilizando la etiqueta em. Por ejemplo, este texto está enfatizado.

3. Texto fuerte

El texto fuerte se puede lograr utilizando la etiqueta strong. Por ejemplo, este texto está fuertemente enfatizado.

4. Texto subrayado

El texto subrayado se puede lograr utilizando la etiqueta u. Por ejemplo, este texto está subrayado.

5. Texto tachado

El texto tachado se puede lograr utilizando la etiqueta strike. Por ejemplo, este texto está tachado.
